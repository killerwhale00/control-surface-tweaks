# Control Surface Tweaks
Control Surface Tweaks is a KSP2 Redux mod that gives you full control over the deflection and deployment limits of all aerodynamic control surfaces.
It allows you to increase the maximum deflection angle of procedural control surfaces beyond the stock limit. also allows you to raise the deployment angle of grid fins and airbrakes beyond their stock limit 
and set a custom default authority limiter separately for all control surfaces under the control surfaces tab 
## All values are configurable through the in-game settings menu
